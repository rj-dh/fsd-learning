CSS Overflow Properties

By Rajnandini Dhongade
Overflow: Visible
This content will overflow outside the box if it is too long. Rajnandini Dhongade demonstrates the visible overflow property.
Overflow: Hidden
This content will be clipped and hidden when it exceeds the box boundaries. Rajnandini Dhongade shows hidden overflow.
Overflow: Auto
Scrollbars will appear automatically if content overflows. Rajnandini Dhongade demonstrates overflow-auto.
Overflow: Scroll
Scrollbars are always visible, regardless of content. Rajnandini Dhongade demonstrates overflow-scroll.
